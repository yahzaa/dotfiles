# dotfiles
Config files

## Install instructions

### Emacs setup
* mkdir ~/.emacs.d/
* ln -s emacs_init.el ~/.emacs.d/init.el
* Launch emacs and wait for package installation to finish.

### i3 setup
* cp i3config ~/.config/i3/config
* CMD + Shift + c to reload config

### vim setup
* cp vimrc ~/.vimrc
